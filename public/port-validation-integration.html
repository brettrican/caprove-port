Port Validation Integration

Complete port management system for CapRover app deployment

Integration Instructions

To integrate this port validation system into CapRover, add the following to your app deployment page:

<script src="/port-validation.js"></script>
<link href="/port-validation.css" rel="stylesheet">

Key Features

  • Real-time Port Validation - Detect conflicts before deployment
  • Smart Suggestions - Get AI-powered port recommendations
  • Visual Indicators - Clear status indicators (valid/conflict/warning)
  • One-click Resolution - Automatically resolve conflicts with alternatives
  • Responsive Design - Works on desktop, tablet, and mobile
  • Accessibility Support - WCAG compliant with keyboard navigation

API Endpoints

  • POST /api/v1/user/appdefinitions/validate-ports - Validate ports before deployment
  • GET /api/v1/user/ports/suggest-for-app - Get port suggestions for new apps
  • GET /api/v1/user/ports/scan - Scan current port usage
  • POST /api/v1/user/ports/validate - Validate specific port
